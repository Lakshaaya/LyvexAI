# LyvexAI Cyber Defense Platform

## Overview

LyvexAI is an AI-powered Security Operations Center (SOC) dashboard designed to monitor, analyze, and respond to cybersecurity threats in real time.

The platform provides:

* Threat Detection
* Risk Scoring
* Blacklist Detection
* Incident Management
* Role-Based Access Control (RBAC)
* Threat Intelligence Dashboard
* Security Reporting
